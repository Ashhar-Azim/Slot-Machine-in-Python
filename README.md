# Slot Machine Game

![Slot Machine](slot_machine.png)

A simple slot machine game with a graphical user interface (GUI) built using Python and Tkinter.

## Table of Contents

- [Features](#features)
- [Requirements](#requirements)

## Features

- Aesthetic and modern GUI.
- Dark mode compatible.
- Deposit money to your balance.
- Choose the number of lines to bet on.
- Set your bet amount per line.
- Spin the slot machine and see the results.
- Win money based on matching symbols and bet.

## Requirements

- Python 3.x
- Tkinter (usually included with Python)
- Pillow (PIL) for image handling (install with `pip install pillow`)
